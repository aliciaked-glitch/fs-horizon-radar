# FS Horizon Radar

This is the GitHub upload package for the UK-first financial services horizon-scanning dashboard.

## Folder structure

- `backend/` = FastAPI API for dashboard data and feedback
- `frontend/` = Next.js dashboard website

## What to upload to GitHub

Upload the **entire `fs-horizon-radar` folder** exactly as it is.

## What happens next

- Deploy `backend` to Railway
- Deploy `frontend` to Vercel
- Add the Railway URL into Vercel as `NEXT_PUBLIC_API_BASE_URL`

## Beginner note

If you are uploading through the GitHub website:
1. Create a new repository.
2. Click **Add file**.
3. Click **Upload files**.
4. Drag the contents of the `fs-horizon-radar` folder into the browser.
5. Click **Commit changes**.
